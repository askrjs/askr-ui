# Foundations

Foundations are the low-level behavior contracts that `askr-ui` components
build on.

## Principle

Foundations define the behavior policy. Components compose them; they do not
reimplement them.

## How to use them

Most applications should compose `askr-ui` components and keep foundations
private to library-level work or new primitive development.
